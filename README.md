# Send Clone Server

This is the server-side application for Send Clone, a file transfer web app.

## Live App

[send-clone.vercel.app](https://send-clone.vercel.app)

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the server:
   ```bash
   node server.js
   ```

## Features
- Real-time file transfer using Socket.IO
- CORS enabled for the live app

---

For the frontend/client, see the `../client` directory.
